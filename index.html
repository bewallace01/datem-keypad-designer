Quick guide — Civil 3D + DAT/EM keypads

If you're new to keypad-driven aerial photogrammetry compilation in Civil 3D + DAT/EM Summit Evolution, this is the 5-minute primer. Skim once, come back when needed.

1. Macro syntax — the prefixes that matter

Every keypad button sends a command string. The DAT/EM keypad executes it as a stream of keystrokes, so the format mirrors what you'd type at the AutoCAD command line:

  • {RET} — the literal token for Enter. Use it between every prompt: -LAYER{RET}SET{RET}BLDG{RET}{RET} is dash-LAYER, Enter, SET, Enter, BLDG, Enter, Enter. The trailing {RET}{RET} exits the LAYER command — forgetting the second one leaves LAYER hanging and eating your next keypress.
  • _ — language-independent command name. _LINE works on French Civil 3D; bare LINE might not. Use it on every AutoCAD command.
  • - — suppresses the dialog popup. LAYER opens the Layer Manager (breaks your collection rhythm). -LAYER runs at the command line. Always use the dash form while digitizing.
  • ' — transparent command. Runs inside the current command without ending it. Mostly for mid-feature OSNAP toggles: '_-osnap{RET}end.
  • No ^C^C, no ;, no real newlines. The AutoCAD CUI conventions (^C^C for cancel, ; for Enter, multi-line macros with \n) do not work on a DAT/EM keypad — ^C^C arrives as literal text and breaks the next prompt. Use only the format above.

2. Three command types, three places they run

  • AutoCAD / Civil 3D keyins like _3DPOLY, -LAYER, _AECCDRAWFEATURELINES — run in the AutoCAD command line. The bulk of your macros.
  • Summit Evolution keywords like Driver, RaiseZ, ZLock — drive Summit itself (the stereo viewer). Case-sensitive — driver won't work, only Driver.
  • DAT/EM Capture commands like AUTOARC3D, PSQR, place cell — come from Capture for AutoCAD. Capture must be loaded in your Civil 3D session or these silently do nothing. Test with AUTOARC3D at the command line — if AutoCAD says "Unknown command," Capture isn't loaded.

3. Patterns you'll use constantly

Switch layer + start linear collection
-LAYER{RET}SET{RET}V-ROAD-PAVE{RET}{RET}AUTOARC3D — DAT/EM Capture's 3D auto-arc, the default for ground-following linear features (roads, curbs, fences, edges, hydro).
Switch layer + collect a building / right-angle feature
-LAYER{RET}SET{RET}V-BLDG{RET}{RET}PSQR2D — DAT/EM "place square" auto-squares 90° corners. Use for buildings, decks, foundations, porches, garages, retaining-wall corners, anything orthogonal.
Switch layer + insert a symbol
-LAYER{RET}SET{RET}V-UTIL-POWR-POLE{RET}{RET}-INSERT{RET}POLE{RET} — point features. Macro ends at the insertion-point prompt; you click in the stereo image to place.
Breakline for surface generation
-LAYER{RET}SET{RET}V-TOPO-BRKL{RET}{RET}_AECCDRAWFEATURELINES — Civil 3D feature line. These can be added to a TIN as actual surface breaklines.
Spot elevation as COGO point
-LAYER{RET}SET{RET}V-TOPO-SPOT{RET}{RET}_AECCCREATEPTMANUAL — surveyable, gets a number, drops on the right layer.
Mid-feature snap toggle
'_-osnap{RET}end — transparent OSNAP set, doesn't interrupt the active drawing command.

4. Stereo workflow tips

  • Z-lock for constant-height features: roof outlines, water surfaces, parking-lot edges. ZLock pins your cursor's elevation; draw the perimeter; ZUnlock when done. Without Z-lock you get a wavy surface no one wants.
  • AutoLevel drops the stereo cursor onto the ground at the current XY. Use when you've drifted up off the surface.
  • Driver is the most-pressed key in the session — it accepts a point at cursor. Put it under your dominant hand, top-left of the keypad.
  • NextStereoPair / PreviousStereoPair let you move between models without leaving the keypad and reaching for the mouse.

5. Pick the right linear collection tool

Three DAT/EM tools cover almost all linear feature work; pick based on the feature's geometry:

  • PSQR2D (place square, 2D) — buildings, decks, foundations, porches, sheds, garages, any right-angle structure. Auto-squares 90° corners as you collect — much faster + cleaner than freehand. Pair with ZLock for roof-level collection (constant Z).
  • AUTOARC3D (3D auto-arc) — curves and organic linears: curbs, road edges, fences, hydro edges, tree lines, sidewalks. Streams a 3D linestring as you drive the stereo cursor.
  • _AECCDRAWFEATURELINES (Civil 3D feature line) — anything that's going into a surface as a breakline (drainage, retaining-wall toes, graded pavement edges). Has elevation-edit tools, plays nice with the TIN.

Rule of thumb: right angles → PSQR2D, organic curves → AUTOARC3D, surface breakline → feature line. Don't bother with _3DPOLY or Bspline — they're worse versions of these for compilation work.

6. Common mistakes that bite

  • Using ^C^C or ; — these are AutoCAD CUI conventions and arrive as literal text on a DAT/EM keypad, breaking the next prompt. Use {RET} tokens and no cancel prefix.
  • Bare LAYER instead of -LAYER — the dialog pops during your collection.
  • Missing trailing {RET}{RET} on -LAYER{RET}SET{RET}NAME{RET}{RET} — LAYER stays open and consumes your next keypress.
  • Case-wrong Summit keywords — driver is silent, only Driver works.
  • Capture commands without Capture loaded — layer switches fine, then AUTOARC3D hits an unknown command and you're staring at an empty command line.
  • Two keypads open at once — one wins the connection, the other quietly drops commands. Close the extra.

7. Using this tool to speed up keypad setup

  1. Drop your project's CAD template (📐 Import CAD file). The tool reads the layer table, pre-selects only the layers that actually have entities in the drawing — usually 30–60 of them out of hundreds defined.
  2. Auto-linked blocks save you the two-step workflow. If your drawing has POLE inserted 46× on V-UTIL-POWR-POLE, the layer button does both — one press switches the layer AND starts -INSERT POLE.
  3. Section headers + color categories let your eye find the right cluster fast. Auto-generated layouts group by category (Roads, Utilities, DTM) with 2-wide colored headers.
  4. Bulk select + recolor / find-replace (Cmd/Ctrl-click cells, then use the floating action bar). Useful for "rename V-ROAD-PAVE to V-PVMT-EDGE across 12 buttons" or "make all the OSNAPs coral."
  5. Custom BMP icons — upload or draw a 24×24 icon for any button. Export as ZIP gives you the .dkf + a bitmaps/ folder ready to drop into C:\DAT-EM\Bitmaps\.
  6. Mark working projects as templates (☆ → ★) so next job starts from your refined layout, not blank.

8. Useful keyboard shortcuts in this designer

  • Cmd/Ctrl + Click a cell → toggle into multi-select.
  • Shift + Click a cell → extend selection to a rectangle.
  • Cmd/Ctrl + Enter in the side panel → save button.
  • Cmd/Ctrl + Z / Cmd/Ctrl + Shift + Z → undo / redo any change (including bulk ops, AI auto-fill, drag-rearrange).
  • Esc → close any modal or the side panel (prompts if unsaved changes).

9. Resources outside this tool

  • DAT/EM downloads page — official PDFs of all Summit Keywords and Capture CallCmds: datem.com/downloads-2/
  • DAT/EM Snapping Guide for Capture for AutoCAD — official OSNAP setup PDF.
  • Civil 3D Help — search for _AECCDRAWFEATURELINES, _AECCCREATEPTMANUAL, _AECCADDSURFBREAKLINES.
